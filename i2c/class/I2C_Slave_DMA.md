## コンストラクタ
##### I2C_Slave_DMA::I2C_Slave_DMA(I2C_HandleTypeDef, T, uint8_t)
> ```c++
> I2C_Slave_DMA(
>     I2C_HandleTypeDef *hi2c,
>     T &data,
>     uint8_t address = 0x00
> );
> ```
> ```c++
> I2C_Slave_DMA(
>     I2C_HandleTypeDef &hi2c,
>     T &data,
>     uint8_t address = 0x00
> );
> ```
> ピンと自局アドレス・連携させた変数を設定します  
> ```c++
> // 例
> uint16_t data;
> I2C_Slave_DMA<uint16_t> slave(hi2c1, data, 0x01);
> I2C_Slave_DMA<uint16_t> slave(&hi2c1, data, 0x01);
> ```
